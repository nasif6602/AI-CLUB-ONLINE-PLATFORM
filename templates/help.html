📊 Dashboard

The dashboard is your home screen. It shows:

  • Club statistics – total members, events, projects, resources.
  • Latest announcements – pinned updates from admins.
  • Upcoming events – next events with dates.
  • Today's Blogs – scrollable stories of blogs published today. Tap to read.
  • Recent Blog Posts – last 4 blog posts.

Who can see it? All logged‑in users.

👥 Members

Browse all club members, search by name/email/skills, and view their public profiles.

  • Click a member's View Profile button to see their blogs, resources, and projects.

Who can access? All authenticated users.

📅 Events

View upcoming and past events. Use the filter buttons (Upcoming, Past, All) to switch.

  • Creating an event – only admins can click + Create and fill in the details.
  • Registering – if an event has a registration link, click Register to open it.

Permissions: Anyone can view, only admins can create events.

📚 Resources

Share and discover AI/ML learning materials. Filter by category (Machine Learning, Python, etc.) and search.

  • Adding a resource – any member can click + Add, provide title, category, link, and description.
  • Click 🔗 Open to visit the resource.

Who can add? All authenticated users.

🚀 Projects

Showcase your AI projects. Include GitHub links, demo links, technologies used, and team members.

  • Add a project – click + Add, fill in the fields, and submit.
  • Viewing – everyone can see projects. Click the GitHub or Demo buttons to open external links.

Permissions: All members can create and view.

📢 Announcements

Important updates from the admin team. Only admins can post, but everyone can read.

  • Creating – admins see a + New button; fill in title and description.

📝 Blog

A space to write and share articles on AI topics. Features a rich editor, reactions, and comments.

Writing a blog

  • Navigate to Blog and click ✍️ Write Blog.
  • Fill in title, optional cover image URL, excerpt, tags (comma‑separated), and content.
  • Check Publish immediately or save as a draft (unpublished).

Reading & interacting

  • Click any blog card to open the full view.
  • Use the emoji reactions (🔥👏💡❤️😮) to react – you can only react once per type.
  • Scroll down to the comments section, write a comment, and post. You can edit or delete your own comments.
  • Admins can delete any comment or blog if necessary.

Views: Each unique user (per browser) counts as one view, preventing self‑inflation.

💬 Forum (Q&A)

A WhatsApp‑style discussion area. Ask questions, share knowledge, and help each other in real time.

Using the forum

  • Select a thread from the left sidebar to open the chat.
  • Type your message at the bottom and press Enter or click Send.
  • Messages from others appear instantly (or within 2 seconds).
  • You can delete your own messages; admins can delete any message or entire thread.

Creating a new thread

  • Click + New in the threads sidebar.
  • Give it a title and choose a category (General, AI, Help, etc.).
  • The thread will appear, and anyone can join the conversation.

Who can participate? All logged‑in members.

💳 Membership

Become an official member to unlock exclusive events, resources, and a member directory.

How to join

  • Go to the Membership page from the sidebar.
  • Scan the QR code or use the UPI ID to pay the membership fee (₹299/year).
  • After payment, enter your transaction ID (UTR) and optionally upload a screenshot.
  • Submit – your request will show as pending until an admin approves it.
  • Once approved, you'll receive a confirmation email and gain access to member‑only areas.

After approval: New sidebar links appear: Member Events, Member Resources, Member Directory.

👤 Profile

Your personal profile displays your info and all content you've created.

  • Edit profile – click ✏️ Edit to update name, branch, year, bio, and skills.
  • Use the tabs below your profile to view your Blogs, Resources, and Projects.

⚙️ Admin Panel

Only accessible to users with the admin role. Provides management tools.

Capabilities

  • Manage members – promote/demote users to admin.
  • Review membership requests – approve or reject payment verifications.
  • Create events & announcements – buttons appear only for admins.
  • Delete inappropriate content – blogs, comments, forum messages.

How to become an admin? Another admin must promote you from the Admin Panel.
